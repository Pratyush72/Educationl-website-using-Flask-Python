# Educationl-website-using-Flask-Python

🚀 Features
🔑 User Authentication – Signup, Login, and Logout system

📚 Notes & Study Material – Upload and download PDFs, categorized by subjects

📝 Project Showcase – Students can share their projects with GitHub integration

💬 Discussion Forum / Chat – For student collaboration and Q&A

🎮 Gamified Rewards – Coin-based reward system for contributions (upload/view notes, projects)

👤 Profile Section – Manage personal details, view earned coins, and membership status

💳 Membership System – Free and Premium tiers with Razorpay/UPI integration

🔒 Secure – Password hashing, OTP verification for signup & reset
